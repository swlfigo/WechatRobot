# WechatRobot
Wechat Mac Message Robot

Base on Wechat Mac Version. 2.6.1 (16871) Tweak

## Usage
```shell
pod install
```

Run `WechatMacMessageRobotFramework.xcworkspace`

(Defalut Wechat App Located in /Application/Wechat)
